1. В файл excludes, с новой строки добавляйте категории которые хотели бы исключить. Категории прописывайте по примеру картинки excludes.
2. Запустите файл main.
3. В процессе работы будет виден прогресс программы.
4. По завершению программы, будет создан файл my.book cо всеми данными.
5. В файле fail_url будут храниться ошибки программы. Ошибку типа 2023-12-07 15:16:28,570 - ERROR - Ошибка при обработке URL https://tehno-dv.ru/vse-tovary/kassy-samoobsluzhivaniya/page-2/: 404 Client Error
нужно игнорировать, она означает, что программа зашла на несуществующую страницу, и продолжила поиск дальше.
6. В файле logging будут храниться логи работы программы.
7. В файле email указывается email на который придет письмо с файлом.
